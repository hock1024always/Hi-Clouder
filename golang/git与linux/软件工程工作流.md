**基础设施即代码（Infrastructure as Code，简称 IaC）** 是一种**用代码来定义、管理和自动部署 IT 基础设施**（如服务器、网络、存储、防火墙等）的实践方法，而不需要手动去搭建环境。

**IaC 的核心思想**

1. 基础设施 = 代码
   - 用 YAML、JSON、HCL（HashiCorp Configuration Language）等格式编写；

1. 版本可控
   - 把 IaC 文件存入 Git，像管理代码一样做 版本控制、代码审查、回滚。
2. 自动化部署
   - 通过工具自动创建/更新/销毁资源，无人工干预。
3. 幂等性（Idempotent）
   - 无论执行多少次，结果都是一样的：
     - 第一次：创建资源；
     - 第二次：发现已存在，不做变更（或只同步差异）。
