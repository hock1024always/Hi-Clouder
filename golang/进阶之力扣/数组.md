## 排序

### 冒泡

```go
func bubbleSort(arr []int) {
	n := len(arr)
	// 外层循环：控制排序轮数，最多需要 n-1 轮（每轮确定一个最大元素的位置）
	for i := 0; i < n-1; i++ {
		// 标记本轮是否发生交换，优化：如果没交换说明已排序完成，可提前退出
		swapped := false
		// 内层循环：每轮比较到 n-1-i 的位置（因为后 i 个元素已排好序）
		for j := 0; j < n-1-i; j++ {
			// 升序：如果前一个元素大于后一个，交换位置
			if arr[j] > arr[j+1] {
				arr[j], arr[j+1] = arr[j+1], arr[j]
				swapped = true
			}
		}
		// 本轮无交换，说明数组已完全有序，直接退出
		if !swapped {
			break
		}
	}
}
```
